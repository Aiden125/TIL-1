# 2장 서블릿

서블릿은 Java EE에 포함된 스펙중 하나로 자바에서 HTTP 요청과 으답을 처리하기 위한 내용들을 담고 있다.

## 서블릿 내부 동작
웹 애플리케이션 컨테이너에서 콘텍스트가 초기화되면 생명주기가 시작된다.
