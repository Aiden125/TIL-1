# GIT 사용 법

## Git 자주 사용하는 명령어

```
git checkout -b [Local Brunch Name] [origin/Remote Brunch Name]
ex -> git checkout -b develop-api-init origin/develop-api-init
```

* remote에 있는 brunch checkout 명령어


## git 핵심 키워드

* commit : 기록을 저장한다
* push : 서버에 보낸다
* pull : 서버에서 내려받는다.
* merge : 두개의  branch를 합친다.
* branch : 각각의 독립된 환경
