= pandoc = == Install == 
brew install pandoc
== pandoc 이란 ? == Haskell 로 만들어진 converter 로 markdown, MediaWiki, textile, HTML, ms 워드(.docx), epub, PDF 등으로 변환이 가능한 유틸리티 [https://www.lesstif.com/pages/viewpage.action?pageId=26083394 출처] Markdown to MediaWiki를 주로 사용 == 사용 목적 == * MediaWiki 작성 문법이 어렵고 위키 작성 생상성이 좋지 않음 * 마크다운 포멧으로 위키 작성하고 pandoc convert를 사용해서 MediaWiki 포맷으로 자동 변경 == Markdown to MediaWiki == === Terminal === 
pandoc file_name.md -f markdown -t mediawiki -s -o file_name.html
